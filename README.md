# Musify Landing Page

This is a simple landing page for "Musify," a music streaming platform. The page is divided into several sections, each with its own content.

## Features

- A navigation bar with links to the Contact, Pricing, and Download sections
- A title section with a heading and two download buttons
- A features section with three feature boxes highlighting the platform's main features
- A testimonials section with a carousel displaying customer testimonials.

## Technologies Used

The page was built using the following technologies:

- HTML
- CSS
- Bootstrap
- Google Fonts
- Font Awesome
